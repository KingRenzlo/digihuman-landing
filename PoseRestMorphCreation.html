Pose→Rest & Morph Creation Guide

How to change rest pose and build morphs in Blender without breaking weights — UE-ready, customer-safe, zero explosions.

Bones and mesh must agree on a single rest position. This guide keeps them in sync.

UE5 Ready Manny/Quinn Compatible No Weight Loss

Part 1 — Changing Rest Pose Safely

Why this is important

When you change a character’s pose or proportions, the skeleton (armature) and the mesh must “agree” on their rest position. If you update one without the other, you get broken weights, stretched limbs, or “exploded” meshes.

This method keeps bones and mesh in sync without losing any weight painting or vertex groups.

  1. Pose the Armature
    Select your Armature → Pose Mode.
    Move bones into the desired proportions or pose you want as the new “rest” position.
  2. Freeze the Mesh to Match
    In Object Mode:
    Select the mesh, then Shift+select the Armature.
    Press Ctrl+A → Visual Geometry to Mesh.
    This bakes the mesh to match the posed bones exactly.
  3. Apply as Rest Pose
    Select the Armature → Pose Mode.
    Go to Pose → Apply → Apply Pose as Rest Pose.
    Now your mesh and armature agree on the new rest position.

✅ No weight loss
✅ No bind mismatch
✅ Fully UE-ready

Part 2 — Creating a Shape Key from a Posed Character

  1. Pose or Adjust the Rig
    Pose the Armature into your desired shape (e.g., stylized proportions, body morph, donor skeleton fit).
  2. Bake the Posed Mesh
    Select mesh → Shift+select Armature.
    Ctrl+A → Visual Geometry to Mesh.
  3. Store as Shape Key
    Select the mesh → Object Data Properties (triangle icon).
    Under Shape Keys, click + to add a new key (this is the “Basis”).
    Click the small ▼ menu → Join as Shapes and choose the posed version.
    You now have a morph slider for that shape.

✅ Perfect for creating morph packs
✅ Works with any pose or donor skeleton
✅ Preserves original mesh

Tips for Digi.Human Users

  • Always keep an untouched backup of your base mesh and rig.
  • Use Edit Mode for proportion changes if you want bones to move without deforming the mesh.
  • Use Visual Geometry to Mesh for any pose-based changes to keep weights perfect.
